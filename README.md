# Calculation Solitaire 🃏🧠

A strategic and engaging **Calculation Solitaire** (also known as *Broken Intervals*) game built with **Flutter**. Unlike luck-based card games, this is a highly intellectual single-player game that requires careful planning and logical thinking.

## 🌟 Key Features
- **Authentic Gameplay:** Faithful reproduction of the "Calculation" solitaire rules, focusing on sequence building.
- **Mental Challenge:** A perfect "brain training" game that rewards strategy over random luck.
- **Global Support:** Fully localized in **50+ languages**, allowing puzzle fans worldwide to enjoy the challenge.
- **Sleek Performance:** Optimized for a smooth and responsive experience on both Android and iOS.

## 💡 How it Works
Players must build four foundation piles starting from Ace, 2, 3, and 4, each increasing by a specific interval (e.g., the 2-pile goes 2, 4, 6, 8...). It is a game of skill and foresight.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Custom algorithms for card sequence validation and state management for complex move tracking.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
